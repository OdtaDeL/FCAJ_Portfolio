# Sharing and Feedback

## 1. Feelings about the program

The First Cloud AI Journey internship program has been an incredible learning experience. I felt both challenged and supported throughout the process of architecting and deploying SpendWise. The combination of cloud architecture theory and hands-on deployment on AWS helped me deeply understand how to operate an enterprise-grade system. The guidance from the AWS team was truly invaluable.

---

## 2. Satisfaction level

I am highly satisfied with the program (9/10). The curriculum was well-structured, covering essential AWS services from networking (VPC) to compute (ECS Fargate) and database management (RDS). The weekly sessions and community events significantly enhanced the learning experience. The only area for improvement would be more deep-dive case studies on cost optimization for fixed-cost services like NAT Gateways or ALB.

---

## 3. Knowledge gained

Throughout the program, I acquired deep knowledge in:

- **Compute & Web Services:** Amazon ECS Fargate (for NestJS), AWS Amplify (for NextJS), and Amazon EC2 (Bastion Host).
- **Networking & Load Balancing:** Designing VPCs with Public/Private Subnets, configuring Application Load Balancers (ALB), and NAT Gateways.
- **Data Management & Security:** Amazon RDS (PostgreSQL), Amazon Cognito, AWS WAF, and AWS Secrets Manager.
- **Monitoring & Operations:** Using Amazon CloudWatch for logs/metrics management and Amazon ECR for container image hosting.
- **Infrastructure as Code (IaC):** Leveraging Terraform to automate infrastructure and manage Security Group rules.

This knowledge was instrumental in building the stable and secure foundation of the SpendWise platform.

---

## 4. Skills improved

Beyond technical skills, I improved my professional soft skills including:

- **System Architecture Design:** Learning to implement a 3-tier architecture to ensure the security of sensitive financial data.
- **Cloud Cost Management:** Analyzing and optimizing AWS budgets (Cloud Cost Optimization).
- **Technical Documentation:** Clearly presenting architecture diagrams and estimated budget sheets.
- **Troubleshooting:** Resolving connectivity issues within private networks and managing environment variable configurations across environments.

---

## 5. Areas for improvement (for the program)

Some suggestions for improving the program:

1. Provide more advanced hands-on labs for VPC Endpoints to optimize NAT costs.
2. Offer more detailed guidance on setting up CI/CD pipelines specifically for Container-based architectures (ECS).
3. Include mentoring sessions focused on "Hardening" database security in a cloud environment.
4. Provide more Terraform-specific resources or translated guides for complex service configurations.

---

## 6. Would you recommend the program to friends? Why?

Absolutely, I would highly recommend the FCJ internship program to anyone aiming for a solid Cloud Engineer career path. The program provides:
- **Real-world Experience:** You don't just learn theory; you build a live system like SpendWise from scratch.
- **Access to Modern Tech:** Hands-on experience with the latest Containerization and Managed Service models on AWS.
- **Systemic Thinking:** A comprehensive view of a project, from infrastructure and security to operational budgeting.

It is an excellent launchpad for any developer looking to transition into a Cloud-focused engineering role.
